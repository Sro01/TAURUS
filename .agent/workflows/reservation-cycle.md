1. 한 날짜의 각 타임 슬롯은 한 팀만 예약 가능
2. 바로 예약을 통해 예약 시 해당 타임 슬롯은 예약 불가능 상태가 됨
3. 미리 예약을 통해 예약 시 해당 타임 슬롯은 예약 대기를 걸어둔 상태가 됨
4. 초기화 시간(매주 일요일 오후 6시)이 되면 주차 교체가 일어남
- 이때 미리 예약 해뒀던 타임 슬롯들의 예약 가능/불가능 상태가 결정됨
- 예약 대기가 걸린 타임슬롯 중에서 딱 1팀만 예약해둔 타임슬롯만이 예약 불가능 상태가 됨 (미리 예약 성공)
- 나머지는 모두 예약 가능 상태가 됨 (미리 예약 실패)
